# EgaleX5 - Password Cracker

This is a tool to crack passwords from ZIP files using a wordlist. The script is obfuscated/encrypted to protect the source code.

## How to Use:
1. Download the executable or use the provided wrapper.
2. Ensure you have a valid wordlist file.
3. Run the tool with the required arguments for the ZIP file and wordlist.
